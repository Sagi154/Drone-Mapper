# High Level Design (HLD) — scaffold

This repository ships **diagram sources** under `docs/diagrams/` and this Markdown outline.

For course submission, export a single **`HLD.pdf`** (per assignment) that includes:

- The UML class diagram (from `class_diagram.puml` or your final version)
- The UML sequence diagram for the main flow (from `sequence_diagram.puml` or your final version)
- Short sections for design considerations, alternatives, and testing approach

Suggested export paths:

- PlantUML → PNG/SVG → paste into Word/Google Docs → export PDF, or use a PlantUML PDF pipeline
- If you use a diagram tool (draw.io), replace the `.puml` sources with your canonical diagrams and still submit **`docs/HLD.pdf`**
